# Análisis de Partidos de la Copa del Mundo
##  Descripción
Análisis exploratorio de datos de todos los partidos de la Copa del Mundo (1930-actualidad) para identificar patrones históricos, tendencias de goles, ventaja de localía y rendimiento de selecciones.

#  Tecnologías
Python (Pandas, Matplotlib, Seaborn)

Jupyter Notebook

Git/GitHub

# Arquitectura
Ingesta: Carga de datos CSV

Procesamiento: Filtrado y transformación

Análisis: Cálculo de métricas estadísticas

Visualización: Generación de gráficos

# Jupyter Notebook
Contiene el análisis completo estructurado en: carga de datos, exploración inicial, análisis temporal, análisis de resultados, ranking de selecciones y conclusiones.

# Resultados Clave
Promedio de goles: Mayor en 1954 (5.38), menor en 1990 (2.21)

Distribución: Victorias local 42.1%, visitante 30.5%, empates 27.4%

Ventaja anfitrión: 65.4% de victorias jugando en casa

Top selecciones: Brasil (5 títulos), Alemania/Italia (4), Argentina (3), Francia (2)

Tendencia: Disminución gradual de goles y creciente equilibrio competitivo
